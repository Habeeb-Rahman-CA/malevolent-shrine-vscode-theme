# Change Log

All notable changes to the "habrmnc-theme" extension will be documented in this file.

## [0.1.0] - 2026-03-19

### ✨ Added
- **Multi-Domain Expansion**: The extension now includes four unique themes:
  - **Malevolent Shrine**: Dark Crimson & Gold aesthetic (Sukuna themed).
  - **Infinite Void**: Deep Purple & Blue high-end aesthetic (Gojo themed).
  - **Divine Collapse**: Celestial Slate & Electric Blue aesthetic.
  - **Null Existence**: A pure monochromatic grayscale experience.
- **Terminal Themes**: Custom ANSI color palettes for each domain theme.
- **Unified UI**: Full interface redesign for sidebar, activity bar, and status bar for all themes.

### 🎨 Fixed/Optimized
- Refined color contrasts for long-term eye comfort.
- Minimalist font color updates for better hierarchy and code focus.

## [0.0.1] - Initial Release
- Initial release of the Malevolent Shrine theme.
